# Todo List Application

A Java-based desktop Todo List application developed to help users manage tasks in an organized and interactive way.  
This project was built using Java Swing while learning GUI development, Object-Oriented Programming, and event-driven programming concepts. The application includes productivity-focused features such as task tracking, progress monitoring, and a built-in Pomodoro timer.

## Features
- Add and delete tasks dynamically
- Mark tasks as completed using checkboxes
- Progress tracking using a progress bar
- Built-in Pomodoro timer with Start, Pause, and Reset functionality
- Interactive completion notifications
- Customized graphical user interface using Java Swing
- Real-time UI updates based on user actions

## Technologies Used
- Java
- Java Swing
- AWT
- Object-Oriented Programming (OOP)
- Event Handling
- Git & GitHub

## Project Structure
- `Todo.java` – Main application logic, GUI components, task handling, and Pomodoro timer
- `AddTask.java` – Task model class used to store and manage individual tasks
